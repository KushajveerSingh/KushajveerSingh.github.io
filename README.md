# KushajveerSingh.github.io

## Markdown docs frontmatter

```
---
title: used in title, next, prev, sidebar
description: by default first line of markdown, appears in meta description
image: cover or thuimbnail image when displaying link to your post
slug: to customize the document url
draft: false, only available in development
unlisted: false, they are hidden in production, not indexed, no sitemap
last_update: override last update manually

id: unique_document_id (default, file path including folders without extension)
pagination_label: text used in next/prev buttons
sidebar_label: text used in sidebar
sidebar_position: (number) sidebar position when using autogenerated
hide_title: to hide the title (defined in frontmatter) to appear in markdown
hide_table_of_contents: false
toc_min_heading_level: 2
toc_max_heading_level: 3
pagination_next: set to "null" without quotes to hide next button
pagination_prev: same as above
---
```
