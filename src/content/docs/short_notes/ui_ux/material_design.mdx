## Accessibility

The latest WCAG guidelines can be found at https://www.w3.org/WAI/standards-guidelines/wcag/.

Increase usability for all users, including those with low vision, blindness, hearing impairments, cognitive impairments, motor impairments, or situational disabilities (such as a broken arm).

List of assistive technology
- **Keyboard**. Hardware or software allowing users to jump from selection to selection in a linear fashion.
