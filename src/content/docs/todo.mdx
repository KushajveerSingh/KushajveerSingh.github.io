## Web projects
Book to showcase the progress of web development

Start with blank canvas in c. now implement html, css, server, http and then move one step at a time. You do not have to discuss technologies in chronological order.
So when dicussing css, you can just finish css/sass/bem/tailwind/css-in-js (assume the person reading the text already has knowledge about the ecosystem). This is not a beginner friendly book.
gituhb
Start with raw HTML/CSS/JS.

Make a project and tell what problems we faced, that led to the next thing like jQuery.

Then make project on this and move the next problem that was solved by the next thing.

This includes everything from webpack, vite. You go implement them yourselves to shwocase how these tools work internally (like nextjs / qwik, ...)
